## 1. Anthropic vs Amazon ($AMZN)

> 凭借卓越的代码生成能力，Anthropic 的 Claude 3.5/3.7 Sonnet 已成为 Manus、Cursor 等头部 AI Agent 的核心底层基础设施。由于代码是打通数字世界各个孤岛的关键中间层，Claude 在解决代码问题上的突破使其在通用 Agent 生态中占据绝对优势地位，这将极大带动其 API 调用量及背后云服务商的算力需求。

置信度: HIGH | 新颖度: HIGH | 可行动性: HIGH

**验证**: ✅ 已验证 (2026-03-20)

**影响路径**

- 技术到应用的传导：Claude 3.7 Sonnet 在 SWE-bench 等代码基准测试中表现优异（准确率达70.3%），使其成为 Cursor（头部AI代码编辑器）和 Manus（全能型AI Agent）等现象级应用的首选底层大模型。
- 应用到生态的传导：代码能力是 AI Agent 操作计算机、调用工具和连接不同软件孤岛的基础。Claude 在代码和 Agentic（智能体）任务上的突破，使其在通用 Agent 生态中占据了绝对的主导地位。
- 生态到算力/云厂商的传导：随着基于 Claude 的 AI Agent 用户量和使用频次激增，Anthropic 的 API 调用量呈指数级增长。作为 Anthropic 的主要投资者和云服务提供商，Amazon AWS（通过 Amazon Bedrock 等）将直接受益于庞大的底层算力租赁和云服务需求激增。

## 2. Manus

> 明星初创公司 Manus 获得 Benchmark 投资，估值达 5 亿美元。但其面临严重的商业模式风险：一是定位过于通用导致缺乏核心心智和网络效应，可能重蹈 Google Wave 的覆辙；二是核心智力高度依赖外部模型（如 Claude），存在 API 波动和成本不可控的风险。预计其下一步必须向垂直应用场景转型以求生存。

## 5. Instacart ($CART) vs OpenAI

> OpenAI 正通过强化学习微调（RFT）技术推动 o1 模型向具备连续行动能力的 Operator 演进，并已深度集成 Instacart 等电商平台实现从搜索到支付的闭环。相较于早期依赖静态工作流的方案，AI 能在动态环境中自主调整策略。这一技术路线表明通用 Agent 正进入系统级操作阶段，有望重塑现有的电商与服务分发入口。

置信度: HIGH | 新颖度: HIGH | 可行动性: HIGH

**影响路径**

- 从技术到产品的传导：RFT技术赋予大模型（如o1系列及CUA模型）更强的逻辑推理与纠错能力，使其能突破静态API限制，在动态网页环境中自主完成点击、输入等连续操作。
